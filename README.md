# ✨ Features

- 📡 WiFi Access Point
- 🎮 Real-time camera control
- ↔️ Pan & Tilt using two servo motors
- 📷 Live video streaming
- 📱 Control directly from a web browser
- 🖨️ Fully 3D printable design
- 🔓 Open-source hardware & software
